# Autonomous AIOps Self-Healing Monitoring System

## Project Overview
This project demonstrates a simplified **AIOps (Artificial Intelligence for IT Operations)** framework that monitors system metrics, detects anomalies using machine learning, and triggers automated alerts and self-healing actions.

The goal of this project is to simulate how modern IT operations platforms use AI to improve system reliability, detect failures early, and automate incident response.

- Real-time system metrics monitoring
